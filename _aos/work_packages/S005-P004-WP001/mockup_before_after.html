WP Scope Legend

WPWhat changesLocale rule
WP001 Data Fields price_chf → price + currency, "Tram" → "Transit", all display reads currency from data Provides data plumbing for per-market currency. No mixed currencies in one table.
WP002 Registry Hardcoded scraper mapping → FQN dynamic registry. No visible UI change. N/A — backend only
WP003 Locale Locale dataclass with direction, html_lang, keywords, labels. get_locale(country) accessor. Defines complete locale per country. WP004 consumes direction/html_lang for RTL rendering.
WP004 HTML i18n BLOCKED ~50 German strings → Hebrew, <html dir="rtl">, bootstrap.rtl.min.css, Module 11 compliance 100% locale rule: IL = all Hebrew + RTL. CH = all German + LTR. No mixing. Must follow AOS Module 11.
