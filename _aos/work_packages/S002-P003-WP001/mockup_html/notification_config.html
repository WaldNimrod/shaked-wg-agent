Notification Configuration

Benachrichtigungseinstellungen sind im Suchprofil eingebettet. Benutzer waehlen welche Kanaele und wie viele (max. 5).

Speicherort: data/profiles/default.json → Feld notifications
Eingebettet im SearchProfile — jedes Profil hat eigene Benachrichtigungseinstellungen mit frei waehlbaren Kanaelen.
SearchProfile "default" — notifications Feld (Channels-Liste)
{
  "profile_id": "default",
  "profile_name": "Shaked Basel WG Search",
  "...": "...",
  "notifications": {
    "digest_max_listings": 5,
    "min_score_threshold": 40,
    "channels": [
      {
        "type": "email",
        "enabled": true,
        "label": "Personal email",
        "recipients": ["shaked@example.com"]
      },
      {
        "type": "telegram",
        "enabled": true,
        "chat_id": "123456789"
      },
      {
        "type": "discord",
        "enabled": false,
        "webhook_url": "https://discord.com/api/webhooks/..."
      },
      {
        "type": "ntfy",
        "enabled": true,
        "topic": "shaked-wg-a8f3k2x9"
      },
      {
        "type": "webhook",
        "enabled": false,
        "url": "https://example.com/hook"
      }
    ]
  }
}
Per-Profil, frei waehlbar: Jedes Suchprofil definiert seine eigene Kanal-Liste (max. 5 Kanaele, beliebige Kombination). Mehrere Kanaele des gleichen Typs sind erlaubt.
Unterstuetzte Kanaele (5 Typen)
Kanal type Kanalspezifische Felder Benutzer-Setup
Email email recipients (string[]) Email-Adresse angeben
Telegram telegram chat_id (string) /start beim Bot, Chat-ID erhalten
Discord discord webhook_url (string) Webhook im Server erstellen
ntfy ntfy topic (string), server_url (optional) ntfy-App installieren, Topic abonnieren
Webhook webhook url (string), headers (optional) URL angeben (IFTTT, Zapier, n8n, ...)
Benoetigte Umgebungsvariablen (nur Email + Telegram)

Discord, ntfy und Webhook benoetigen keine Umgebungsvariablen — URLs sind im Kanal-Config enthalten.

Variable Kanal Beschreibung Beispiel
SMTP_HOST Email SMTP-Server Hostname smtp.gmail.com
SMTP_PORT Email SMTP-Server Port 587
SMTP_USER Email SMTP-Benutzername bot@example.com
SMTP_PASS Email SMTP-Passwort / App-Passwort xxxx-xxxx-xxxx
SMTP_FROM Email Absender-Adresse shaked-wg@example.com
TELEGRAM_BOT_TOKEN Telegram Bot-Token von @BotFather 123456:ABC-DEF...
