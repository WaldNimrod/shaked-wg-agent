## 1. Scope Reminder

Create a ManualFacebookScraper — a BaseScraper subclass that reads raw Facebook group posts from a local JSON file, sends each post to an LLM (Claude/OpenAI) for structured Hebrew extraction, filters non-rental posts, strips PII, deduplicates against existing listings, and outputs ScrapedListing entries. 8. **Main dispatch function** with error handling per the LOD200 failure behavior matrix (7 modes, 1:1 mapping):

   ```python
   def parse_rental_post(text: str, group_name: str = "", post_id: str = "") -> dict | None:
       """Parse a Hebrew rental post. Returns dict or None. NEVER raises."""
       
       # MODE 1: No provider configured (run-level)
       if not _LLM_PROVIDER:
           # Logged at scraper level, not here (scraper checks check_llm_config() first)
           return None
       
       # MODE 2: No API key configured (run-level)
       if _LLM_PROVIDER == "claude" and not _ANTHROPIC_KEY:
           return None
       if _LLM_PROVIDER == "openai" and not _OPENAI_KEY:
           return None
       
       call_fn = _call_claude if _LLM_PROVIDER == "claude" else _call_openai
       
       try:
           return call_fn(text, group_name)
       
       except Exception as e:
           err_str = str(e).lower()
           
           # MODE 4: API rate limit (429) — post-level, retry once
           if "rate_limit" in err_str or "429" in err_str or "rate limit" in err_str:
               logger.warning("LLM rate-limited for post %s, retrying in 2s", post_id)
               time.sleep(2)
               try:
                   return call_fn(text, group_name)
               except Exception:
                   logger.warning("LLM rate-limited after retry, skipping post %s", post_id)
                   return None
           
           # MODE 3: API timeout — post-level, skip
           if "timeout" in err_str or "timed out" in err_str:
               logger.warning("LLM timeout for post %s, skipping", post_id)
               return None
           
           # MODE 6: API error (500, network) — post-level, skip
           logger.error("LLM API error for post %s: %s", post_id, e)
           return None
   ```

   **MODE 5 (malformed response)** is handled in `_parse_response()` (see item 7 above) — returns None with WARNING including first 200 chars of raw response.

   **Failure behavior matrix — LOD200 1:1 mapping verification:**

   | LOD200 Mode | LOD200 Scope | LOD200 Behavior | LOD400 Implementation | LOD400 Log Level |

   | No API key | Run-level | `fetch_listings()` returns `[]` | `check_llm_config()` check in scraper → return `[]` | WARNING: "No LLM API key; facebook-manual scraper disabled" |
   | No provider | Run-level | `fetch_listings()` returns `[]` | `check_llm_config()` check in scraper → return `[]` | WARNING: "No LLM provider; facebook-manual scraper disabled" |

3. **`fetch_listings()` method** — main pipeline:
   ```python
   def fetch_listings(self) -> list[ScrapedListing]:
       # Step 1: Check LLM config (distinct warnings per matrix modes 1 & 2)
       if not _LLM_PROVIDER:
           logger.warning("No LLM provider configured (SHAKED_LLM_PROVIDER); facebook-manual scraper disabled")
           return []
       if not check_llm_config():
           logger.warning("No LLM API key for provider '%s'; facebook-manual scraper disabled", _LLM_PROVIDER)
           return []

       # Step 2: Read input file
       posts = self._read_input_file()
       if not posts:
           return []

       # Step 3: Parse each post
       listings: list[ScrapedListing] = []
       seen_ids: set[str] = set()
       failed_count = 0
       
       for post in posts:
           post_id = post.get("post_id", "")
           text = post.get("text", "")
           
           # Validate required fields
           if not post_id or not text or len(text) < 10:
               logger.warning("Skipping post: missing required field (post_id=%s)", post_id)
               continue
           
           # Within-file dedup by post_id
           if post_id in seen_ids:
               logger.warning("Duplicate post_id %s, skipping", post_id)
               continue
           seen_ids.add(post_id)
           
           # LLM parse
           parsed = parse_rental_post(text, post.get("group_name", ""), post_id=post_id)
           if parsed is None:
               failed_count += 1
               continue
           if not parsed.get("is_rental_offer", False):
               continue
           
           listing = self._to_scraped_listing(post, parsed)
           listings.append(listing)
       
       if failed_count == len(posts) and len(posts) > 0:
           logger.error("All %d posts failed LLM parsing", len(posts))
       
       return listings
   ```

4. **`_read_input_file()` method:**

   def _read_input_file(self) -> list[dict]:
       if not self.input_path.exists():

           return []
       try:
           with open(self.input_path, encoding="utf-8") as f:
               data = json.load(f)
           if not isinstance(data, list):
               logger.error("Input file is not a JSON array: %s", self.input_path)
               return []
           return data
       except json.JSONDecodeError as e:
           logger.error("Invalid JSON in %s: %s", self.input_path, e)
           return []

   def _validate_post(self, post: dict, seen_ids: set[str]) -> str | None:
       """Validate a single post against normative schema. Returns error message or None."""
       post_id = post.get("post_id", "")
       text = post.get("text", "")
       
       # post_id: REQUIRED, non-empty, unique
       if not post_id:
           return "missing required field: post_id"
       if post_id in seen_ids:
           return f"duplicate post_id: {post_id}"
       
       # text: REQUIRED, non-empty, min 10 chars
       if not text or len(text.strip()) < 10:
           return f"text too short or missing (post_id={post_id})"
       
       # group_name: OPTIONAL, default ""
       if "group_name" not in post:
           post["group_name"] = ""
       
       # group_url: OPTIONAL, informational only — no validation enforced
       # (invalid URLs are harmless — field is not used in output)
       
       # author_name: OPTIONAL, default "" — NEVER propagated to output
       # (stripped in _to_scraped_listing, not stored in any ScrapedListing field)
       
       # posted_at: OPTIONAL, ISO 8601 if present, default null
       posted_at = post.get("posted_at")
       if posted_at:
           try:
               datetime.fromisoformat(posted_at.replace("Z", "+00:00"))
           except (ValueError, TypeError):
               logger.warning("Invalid posted_at for %s: %s — treating as null", post_id, posted_at)
               post["posted_at"] = None
       
       # has_images: OPTIONAL, default false — coerced to boolean
       post["has_images"] = bool(post.get("has_images", False))
       
       # raw_url: OPTIONAL, stored as-is in direct_url — no URL validation
       # (broken URLs are acceptable; url_status field handles this)
       
       return None  # validation passed
   ```

   def _to_scraped_listing(self, post: dict, parsed: dict) -> ScrapedListing:
       # PII stripping: remove phone numbers from summary
       summary = self._strip_pii(parsed)
       
       return ScrapedListing(
           source=self.source_id,
           source_listing_id=post.get("post_id", ""),
           source_search_url=str(self.input_path),
           title=self._build_title(parsed),
           price=parsed.get("price_ils"),
           currency="ILS",
           country="IL",
           available_from=parsed.get("available_from"),
           location_text=self._build_location(parsed),
           district=parsed.get("neighborhood") or parsed.get("city", ""),
           summary=summary,
           direct_url=post.get("raw_url", ""),
           url_status="direct" if post.get("raw_url") else "none",
           posted_date=self._parse_posted_at(post.get("posted_at")),
           vegan_signal=self._detect_vegan_signal(parsed),
           roommate_signal="",
       )
   ```

6. **`_strip_pii()` method** — remove phone numbers from text:
   ```python
   _PHONE_PATTERN = re.compile(r"0\d{1,2}[-.\s]?\d{3}[-.\s]?\d{4}")
   
   def _strip_pii(self, parsed: dict) -> str:
       """Build summary from parsed data, stripping phone numbers."""
       parts = []
       if parsed.get("property_type"):
           parts.append(parsed["property_type"])
       if parsed.get("rooms"):
           parts.append(f"{parsed['rooms']} rooms")
       if parsed.get("city"):
           parts.append(parsed["city"])
       if parsed.get("key_features"):
           parts.append(", ".join(parsed["key_features"][:5]))
       summary = " · ".join(parts)
       return self._PHONE_PATTERN.sub("[phone removed]", summary)
   ```

7. **Helper methods:**
   ```python
   def _build_title(self, parsed: dict) -> str:
       ptype = parsed.get("property_type", "listing")
       rooms = parsed.get("rooms", "")
       city = parsed.get("city", "")
       if rooms:
           return f"{ptype} {rooms} חדרים ב{city}" if city else f"{ptype} {rooms} חדרים"
       return f"{ptype} ב{city}" if city else ptype
   
   def _build_location(self, parsed: dict) -> str:
       parts = [parsed.get("street", ""), parsed.get("city", "")]
       return ", ".join(p for p in parts if p)
   
   def _parse_posted_at(self, raw: str | None) -> str | None:
       if not raw:
           return None
       try:
           dt = datetime.fromisoformat(raw.replace("Z", "+00:00"))
           return dt.strftime("%Y-%m-%d")
       except (ValueError, TypeError):
           logger.warning("Invalid posted_at format: %s", raw)
           return None
   
   def _detect_vegan_signal(self, parsed: dict) -> str:
       features = parsed.get("key_features", [])
       for f in features:
           if any(kw in f.lower() for kw in ("vegan", "טבעוני", "צמחוני")):
               return f
       return ""
   ```

   ```python
   def _deduplicate_against_existing(self, listings: list[ScrapedListing]) -> list[ScrapedListing]:
       """Dedup against existing listings.json using fuzzy match."""
       existing_path = Path("data/listings.json")
       if not existing_path.exists():
           return listings
       
       try:
           with open(existing_path) as f:
               existing = json.load(f)
       except (json.JSONDecodeError, FileNotFoundError):
           return listings
       
       unique: list[ScrapedListing] = []
       for listing in listings:
           if self._is_duplicate(listing, existing):
               logger.debug("Cross-source duplicate: %s", listing.source_listing_id)
               continue
           unique.append(listing)
       return unique
   
   def _is_duplicate(self, listing: ScrapedListing, existing: list[dict]) -> bool:
       """Check if listing matches an existing one.
       
       Two-tier match:
         1. Same source + same source_listing_id → exact duplicate.
         2. Cross-source fuzzy: location_text + price(±10%) → probable duplicate.
       
       Note: ScrapedListing has no `rooms` field in the current base model,
       so cross-source dedup uses the two dimensions available on the dataclass
       (location_text, price). This is deliberately conservative — if both
       location and price match within 10%, the listing is treated as duplicate.
       """
       for ex in existing:
           # Tier 1: Same source, same ID → exact duplicate
           if ex.get("source") == self.source_id and ex.get("source_listing_id") == listing.source_listing_id:
               return True
           
           # Tier 2: Cross-source fuzzy — location + price(±10%)
           # Both dimensions required; if either is null/missing → no match
           
           # Price match (±10%)
           ex_price = ex.get("price")
           if listing.price and ex_price:
               price_close = abs(listing.price - ex_price) <= listing.price * 0.10
           else:
               price_close = False
           
           # Location match (case-insensitive substring)
           loc_match = (
               listing.location_text and ex.get("location_text") and
               listing.location_text.lower() in ex.get("location_text", "").lower()
           )
           
           if price_close and loc_match:
               return True
       
       return False
   ```

3. **Integration into `fetch_listings()`** — after the parsing loop, before return:
   ```python
   # After: listings = [... parsed results ...]
   listings = self._deduplicate_against_existing(listings)
   return listings
   ```

## 5. Error Handling

| Error Case | Where | Expected Behavior |
|------------|-------|-------------------|
| No LLM provider env var | `fetch_listings()` | Log WARNING, return `[]` |
| No API key env var | `fetch_listings()` | Log WARNING, return `[]` |
| Input file missing | `_read_input_file()` | Log INFO, return `[]` |
| Input file invalid JSON | `_read_input_file()` | Log ERROR, return `[]` |
| Input file is not array | `_read_input_file()` | Log ERROR, return `[]` |
| Post missing `post_id` | `fetch_listings()` loop | Skip row, log WARNING, continue |
| Post `text` < 10 chars | `fetch_listings()` loop | Skip row, log WARNING, continue |
| Duplicate `post_id` | `fetch_listings()` loop | Skip duplicate, log WARNING, continue |
| LLM API timeout | `parse_rental_post()` | Return None, post skipped |
| LLM rate limit (429) | `parse_rental_post()` | Retry 1x with 2s delay, then None |
| LLM malformed response | `_parse_response()` | Log WARNING with first 200 chars, return None |
| LLM API error (500/network) | `parse_rental_post()` | Return None, post skipped |
| All posts fail LLM | `fetch_listings()` | Log ERROR, return `[]` |

**Invariant:** `fetch_listings()` NEVER raises an exception. It always returns `list[ScrapedListing]`.
