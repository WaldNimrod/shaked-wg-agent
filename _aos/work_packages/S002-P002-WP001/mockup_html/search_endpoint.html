POST

/search

Startet eine neue WG-Suche für das angegebene Suchprofil und gibt einen Zusammenfassungsbericht zurück.

Authentifizierung

X-API-Key Header erforderlich.

Request Body
FeldTypPflichtBeschreibung
profile_idstringJa*Suchprofil-Kennung, z.B. "default"
city_idstringNeinDeprecated Alias — wird intern zum default-Profil der Stadt aufgelöst
* Entweder profile_id oder city_id muss angegeben werden. profile_id wird bevorzugt.

Beispiel:

{
  "profile_id": "default"
}
Beispiel-Request
curl -X POST http://localhost:8000/search \
  -H "Content-Type: application/json" \
  -H "X-API-Key: dein-api-schluessel" \
  -d '{"profile_id": "default"}'
