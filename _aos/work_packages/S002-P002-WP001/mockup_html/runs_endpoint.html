GET

/runs

Gibt eine paginierte Liste vergangener Suchdurchläufe zurück.

Authentifizierung

X-API-Key Header erforderlich.

Query-Parameter
ParameterTypPflichtDefaultBeschreibung
profile_idstringNein—Filtert nach Suchprofil-Kennung
city_idstringNein—Deprecated Filtert nach Stadt
limitintegerNein20Maximale Anzahl Ergebnisse (max 100)
offsetintegerNein0Offset für Paginierung
Beispiel-Request
curl "http://localhost:8000/runs?profile_id=default&limit=5" \
  -H "X-API-Key: dein-api-schluessel"
