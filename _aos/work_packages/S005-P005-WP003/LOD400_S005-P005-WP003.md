   ```python
   class EmailFacebookScraper(BaseScraper):
       """Facebook email notification scraper — parses FB group emails via IMAP or .eml files."""
       
       _FB_SENDER = "notification@facebookmail.com"
       _MIN_SNIPPET_LENGTH = 50
   ```

2. **Constructor:**
   ```python
   def __init__(self, source_id: str, search_url: str, city: CityDefinition):
       super().__init__(source_id, search_url, city)
       self._is_imap = search_url.startswith("imap://")
       self._processed_ids = self._load_processed_ids()
   ```

3. **`fetch_listings()` method:**
   ```python
   def fetch_listings(self) -> list[ScrapedListing]:
       # Step 1: Check LLM config (required for parsing)
       if not check_llm_config():
           logger.warning("No LLM API key; facebook-email scraper disabled")
           return []
       
       # Step 2: Get email messages
       if self._is_imap:
           messages = self._fetch_imap()
       else:
           messages = self._read_eml_files()
       
       if not messages:
           return []
       
       # Step 3: Extract post snippets from emails
       snippets: list[dict] = []
       for msg in messages:
           msg_id = msg.get("message_id", "")
           
           # Dedup layer 1: message-ID
           if msg_id in self._processed_ids:
               continue
           
           extracted = self._parse_email_html(msg["html_body"], msg.get("subject", ""))
           for snippet in extracted:
               snippet["message_id"] = msg_id
               snippet["email_date"] = msg.get("date", "")
               snippets.append(snippet)
           
           # Mark as processed
           self._processed_ids.add(msg_id)
       
       # Step 4: Dedup layers 2-3
       snippets = self._dedup_snippets(snippets)
       
       # Step 5: LLM parse each snippet
       listings: list[ScrapedListing] = []
       for snippet in snippets:
           if len(snippet.get("text", "")) < self._MIN_SNIPPET_LENGTH:
               logger.debug("Snippet too short (%d chars), skipping", len(snippet.get("text", "")))
               continue
           
           parsed = parse_rental_post(snippet["text"], snippet.get("group_name", ""))
           if parsed is None:
               continue
           if not parsed.get("is_rental_offer", False):
               continue
           
           listing = self._to_scraped_listing(snippet, parsed)
           listings.append(listing)
       
       # Step 6: Save processed IDs
       self._save_processed_ids()
       
       return listings
   ```

### 2.2 IMAP Connection & Fetching

**File:** `shaked_wg_agent/scrapers/facebook_email.py` (same file)

**What to implement:**

1. **IMAP credential reading (env-vars ONLY):**
   ```python
   def _get_imap_config(self) -> tuple[str, str, str, str] | None:
       """Return (host, user, password, folder) or None if not configured."""
       host = os.environ.get("SHAKED_EMAIL_HOST", "")
       user = os.environ.get("SHAKED_EMAIL_USER", "")
       password = os.environ.get("SHAKED_EMAIL_PASS", "")
       
       if not all([host, user, password]):
           logger.warning("IMAP not configured: SHAKED_EMAIL_HOST/USER/PASS required")
           return None
       
       # Extract folder from search_url: imap://env/INBOX → "INBOX"
       folder = self.search_url.split("/")[-1] if "/" in self.search_url else "INBOX"
       return (host, user, password, folder)
   ```

2. **IMAP fetch method:**
   ```python
   def _fetch_imap(self) -> list[dict]:
       """Connect to IMAP, fetch unread Facebook notification emails."""
       config = self._get_imap_config()
       if config is None:
           return []
       
       host, user, password, folder = config
       messages = []
       
       try:
           import imaplib
           import email as email_lib
           
           conn = imaplib.IMAP4_SSL(host, timeout=30)
           try:
               conn.login(user, password)
               conn.select(folder)
               
               # Search for unread emails from Facebook
               _, msg_nums = conn.search(None, '(UNSEEN FROM "facebookmail.com")')
               if not msg_nums[0]:
                   return []
               
               for num in msg_nums[0].split():
                   _, msg_data = conn.fetch(num, "(RFC822)")
                   raw_email = msg_data[0][1]
                   msg = email_lib.message_from_bytes(raw_email)
                   
                   parsed = {
                       "message_id": msg.get("Message-ID", ""),
                       "subject": msg.get("Subject", ""),
                       "date": msg.get("Date", ""),
                       "html_body": self._extract_html_body(msg),
                   }
                   messages.append(parsed)
                   
                   # Mark as read
                   conn.store(num, "+FLAGS", "\\Seen")
               
               conn.close()
           finally:
               conn.logout()
       
       except imaplib.IMAP4.error as e:
           logger.error("IMAP error: %s", e)
       except Exception as e:
           logger.error("IMAP connection failed: %s", e)
       
       return messages
   ```

3. **HTML body extraction:**
   ```python
   def _extract_html_body(self, msg) -> str:
       """Extract HTML body from email message (multipart or simple)."""
       if msg.is_multipart():
           for part in msg.walk():
               if part.get_content_type() == "text/html":
                   charset = part.get_content_charset() or "utf-8"
                   return part.get_payload(decode=True).decode(charset, errors="replace")
       elif msg.get_content_type() == "text/html":
           charset = msg.get_content_charset() or "utf-8"
           return msg.get_payload(decode=True).decode(charset, errors="replace")
       return ""
   ```

### 2.4 Facebook Email HTML Parsing

**File:** `shaked_wg_agent/scrapers/facebook_email.py` (same file)

**What to implement:**

1. **HTML parser for Facebook notification emails:**
   ```python
   def _parse_email_html(self, html: str, subject: str) -> list[dict]:
       """Extract post snippets from Facebook notification email HTML.
       
       Returns list of dicts with keys: text, group_name, group_url, post_url
       """
       if not html:
           return []
       
       soup = BeautifulSoup(html, "lxml")
       snippets = []
       
       # Detect email type from subject
       group_name = self._extract_group_name(subject)
       
       # Strategy 1: Look for post content blocks
       # Facebook uses nested tables; post text is typically in <td> with specific styling
       for block in self._find_post_blocks(soup):
           text = block.get("text", "").strip()
           if text:
               snippets.append({
                   "text": text,
                   "group_name": group_name,
                   "group_url": block.get("group_url", ""),
                   "post_url": block.get("post_url", ""),
               })
       
       return snippets
   ```

2. **Post block extraction** — Facebook email HTML uses nested tables with tracking pixels. Key selectors:
   ```python
   def _find_post_blocks(self, soup) -> list[dict]:
       """Find post content blocks in Facebook email HTML."""
       blocks = []
       
       # Primary: links to facebook.com/groups/.../posts/...
       for link in soup.find_all("a", href=True):
           href = link.get("href", "")
           if "/groups/" in href and "/posts/" in href:
               # The post text is typically in the parent or sibling <td>
               parent_td = link.find_parent("td")
               if parent_td:
                   text = parent_td.get_text(separator="\n", strip=True)
                   # Remove "See post" / "View post" link text
                   text = re.sub(r"\b(See|View)\s+(post|more)\b", "", text, flags=re.IGNORECASE).strip()
                   if text:
                       blocks.append({
                           "text": text,
                           "post_url": self._clean_fb_url(href),
                           "group_url": "",
                       })
       
       # Fallback: if no post links found, try generic content blocks
       if not blocks:
           for td in soup.find_all("td"):
               text = td.get_text(strip=True)
               if len(text) > 80 and any(kw in text for kw in ("חדר", "דירה", "להשכרה", "שכירות", "room", "apartment")):
                   blocks.append({
                       "text": text,
                       "post_url": "",
                       "group_url": "",
                   })
       
       return blocks
   
   def _extract_group_name(self, subject: str) -> str:
       """Extract group name from email subject."""
       # Patterns: "New post in GROUP", "N new posts in GROUP", "Popular posts in GROUP"
       patterns = [
           r"(?:New post|new posts|Popular posts) in (.+)",
           r"פוסט חדש בקבוצה (.+)",
           r"פוסטים חדשים בקבוצה (.+)",
       ]
       for pattern in patterns:
           m = re.search(pattern, subject, re.IGNORECASE)
           if m:
               return m.group(1).strip()
       return subject
   
   def _clean_fb_url(self, url: str) -> str:
       """Remove Facebook tracking params from URL."""
       # FB wraps URLs in tracking redirects; extract the actual URL
       if "l.facebook.com/l.php" in url:
           parsed = urllib.parse.urlparse(url)
           params = urllib.parse.parse_qs(parsed.query)
           if "u" in params:
               return params["u"][0]
       return url.split("?")[0]  # Strip query params
   ```

**Acceptance criteria:**

### 2.5 Deduplication Pipeline

**File:** `shaked_wg_agent/scrapers/facebook_email.py` (same file)

**What to implement:**

1. **Processed email ID management:**
   ```python
   _PROCESSED_IDS_FILE = Path("data/facebook/processed_email_ids.json")
   
   def _load_processed_ids(self) -> set[str]:
       if self._PROCESSED_IDS_FILE.exists():
           try:
               with open(self._PROCESSED_IDS_FILE) as f:
                   return set(json.load(f))
           except (json.JSONDecodeError, TypeError):
               return set()
       return set()
   
   def _save_processed_ids(self) -> None:
       self._PROCESSED_IDS_FILE.parent.mkdir(parents=True, exist_ok=True)
       with open(self._PROCESSED_IDS_FILE, "w") as f:
           json.dump(sorted(self._processed_ids), f, indent=2)
   ```

2. **Within-batch dedup (layers 2-3):**
   ```python
   def _dedup_snippets(self, snippets: list[dict]) -> list[dict]:
       """Dedup within batch: URL exact match (P2) and text hash (P3)."""
       seen_urls: set[str] = set()
       seen_hashes: set[str] = set()
       unique = []
       
       for s in snippets:
           # P2: URL exact match
           url = s.get("post_url", "")
           if url and url in seen_urls:
               continue
           if url:
               seen_urls.add(url)
           
           # P3: Text hash
           text_hash = hashlib.sha256(s.get("text", "").strip().encode()).hexdigest()
           if text_hash in seen_hashes:
               continue
           seen_hashes.add(text_hash)
           
           unique.append(s)
       
       return unique
   ```

3. **Cross-source dedup (layer 4)** is handled naturally: the run pipeline merges listings from all sources and the existing dedup logic in the orchestrator applies fuzzy matching.

- [ ] AC-20: Two snippets with the same `post_url` → only first is kept.
- [ ] AC-21: Two snippets with identical text but different URLs → only first is kept (text hash dedup).
- [ ] AC-22: `processed_email_ids.json` is persisted after run and loaded on next run.

## 6. Out of Scope (Explicit)

1. **Email account creation/configuration** — user sets up their own email + FB notifications
2. **Gmail API / OAuth** — IMAP is simpler and universal
3. **Push/webhook email** — polling via IMAP is sufficient for 2x/day scans
4. **Fetching full FB posts from truncated snippets** — would require FB access
5. **Automated notification subscription** — user manages FB notification settings
6. **Non-Facebook email notifications** — only facebookmail.com sender
7. **Manual post input** — that is WP002 scope
