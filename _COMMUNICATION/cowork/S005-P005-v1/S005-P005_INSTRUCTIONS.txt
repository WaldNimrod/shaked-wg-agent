=== PROJECT ===

shaked-wg-agent is a Python CLI application that scrapes rental listings from multiple platforms, scores them by relevance for a specific user profile, and delivers results. The codebase supports both Swiss (CHF) and Israeli (ILS) markets.

Key architecture:
  data/cities/{city_id}.json -> config.py (CityDefinition) -> runner.py -> scrapers
  data/profiles/{profile_id}.json -> config.py (SearchProfile) -> scorer.py
  data/sources.json -> config.py (SourceDefinition) -> runner.py -> _resolve_class() via importlib

ScrapedListing fields: source, source_listing_id, title, price (int|None), currency (str), country (str), available_from, location_text, district, transit_match_lines, roommate_signal, vegan_signal, summary, direct_url, url_status, posted_date.

=== SCRAPER PATTERN (reference: flatfox.py, homeless.py) ===

A scraper subclasses BaseScraper and implements fetch_listings() -> list[ScrapedListing]:
  1. __init__(self, source_id, search_url, city) — called by runner
  2. fetch_listings() — HTTP/Playwright/file-based, parse data, return ScrapedListing list
  3. Each listing sets: price=..., currency=city.currency, country=city.country
  4. Never raises — returns [] on any error
