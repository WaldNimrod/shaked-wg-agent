## 2. Technical Specification

### 2.1 Site Technical Profile

**Source:** https://www.homeless.co.il
**Board URL:** https://www.homeless.co.il/rent/
**Technology:** Server-rendered HTML (ASP.NET Web Forms). Listings are in the initial HTML — no client-side rendering needed.
**Encoding:** UTF-8 with Hebrew text.

**Anti-bot: Cloudflare Managed Challenge (CRITICAL)**

| Method | Result |
|--------|--------|
| Plain `requests.get()` (any headers) | **403** — Cloudflare challenge page |
| Playwright headless (default) | **200** for `/rent/` board pages; **403** for filtered/detail pages |
| Playwright headless + stealth (`navigator.webdriver` removal) | **200** for all pages |

**Strategy:** Use Playwright headless with `navigator.webdriver` removed (stealth). This matches the `wgzimmer_pw.py` pattern already used in this project. The board page HTML contains all listing data — no need to visit individual detail pages.

**URL patterns:**

| Pattern | Example | Description |
|---------|---------|-------------|
| Base rent board | `/rent/` | All rentals, page 1 |
| Pagination | `/rent/2`, `/rent/3` | Page N |
| Area filter | `/rent/inumber1=AREA_ID` | Filter by area ID |
| City filter | `/rent/city=CITYNAME` | Filter by city (Hebrew) |
| Combined | `/rent/inumber1=168$$inumber4=7` | Area + room filter |
| Listing detail | `/rent/viewad,ADID.aspx` | Full listing page (NOT NEEDED — board has all data) |

**Area IDs (inumber1):** Integer identifiers for geographic areas. The builder must determine the correct area ID(s) for the Pardes Hanna region by inspecting the area dropdown/links on the rent page. Look for חדרה (Hadera), פרדס חנה-כרכור, or the broader Sharon/Hadera region. If no exact area ID covers the target region, fetch the base `/rent/` board and filter listings client-side.

### 2.2 HTML Structure

**Listing table:** Listings are in an HTML `<table>` with rows identified by `id="ad_NNNNNN"` where NNNNNN is the numeric ad ID. Each row has attribute `type="ad"`.

**Table columns (left to right):**

| Column | Content | Maps to |
|--------|---------|---------|
| 1 | Checkbox (`selectionarea`) | (skip) |
| 2 | Thumbnail image (`PictureDisplayOnBoard`) | (skip) |
| 3 | Property type (דירה, וילה, etc.) | → title prefix |
| 4 | City (עיר) | → location_text |
| 5 | Neighborhood (שכונה) | → district |
| 6 | Street (רחוב) | → location_text |
| 7 | Rooms (חדרים) | → summary |
| 8 | Floor (קומה) | → summary |
| 9 | Price (מחיר) — format "N,NNN ₪" | → price (parse int) |
| 10 | Entry date (תאריך כניסה) — "מיידי" or DD/MM/YYYY | → available_from |
| 11 | Update date (עדכון) — DD/MM/YYYY | → posted_date |

**Example row HTML:**

### 2.3 Scraper Class — `shaked_wg_agent/scrapers/homeless.py`

**What to implement:**

```python
class HomelessScraper(BaseScraper):
    """Scraper for homeless.co.il Israeli rental classifieds.

    Uses Playwright to bypass Cloudflare managed challenge.
    Follow wgzimmer_pw.py pattern for browser lifecycle.
    """

    MAX_PAGES = 5  # Safety limit on pagination
    _WAIT_SECONDS = 8  # Wait for Cloudflare challenge to resolve
```

**Constructor:** Inherits from `BaseScraper.__init__(source_id, search_url, city)`. No additional init needed.

**Method: `fetch_listings()`**

Follow the `wgzimmer_pw.py` pattern:

```python
def fetch_listings(self) -> list[ScrapedListing]:
    try:
        from playwright.sync_api import sync_playwright
    except ImportError:
        return []

    results: list[ScrapedListing] = []
    try:
        with sync_playwright() as pw:
            browser = pw.chromium.launch(headless=True)
            try:
                ctx = browser.new_context(
                    locale="he-IL",
                    user_agent="Mozilla/5.0 ..."  # standard Chrome UA
                )
                page = ctx.new_page()
                # Remove webdriver detection (Cloudflare stealth)
                page.add_init_script("Object.defineProperty(navigator, 'webdriver', {get: () => undefined})")

                for page_num in range(1, self.MAX_PAGES + 1):
                    url = self._page_url(page_num)
                    page.goto(url, wait_until="domcontentloaded")
                    page.wait_for_timeout(self._WAIT_SECONDS * 1000)
                    html = page.content()
                    soup = self._soup(html)
                    rows = self._parse_listing_rows(soup)
                    if not rows:
                        break
                    results.extend(rows)
                    if not self._has_next_page(soup):
                        break
            finally:
                browser.close()
    except Exception:
        pass  # Return whatever we collected
    return results
```

**Method: `_page_url(page_num)` → `str`**

```python
def _page_url(self, page_num: int) -> str:
    if page_num == 1:
        return self.search_url
    base = self.search_url.rstrip("/")
    return f"{base}/{page_num}"
```

**Method: `_parse_listing_rows(soup)` → `list[ScrapedListing]`**

For each `<tr>` with `id` matching pattern `ad_\d+`:
1. Extract ad ID from row id attribute: `id="ad_738229"` → `"738229"`
2. Extract `<td>` cells in order (skip first 2: checkbox, image)
3. Cells 3-11: property_type, city, neighborhood, street, rooms, floor, price, entry_date, update_date
4. Parse price: strip `₪`, remove commas, convert to `int`. If empty/non-numeric → `None`
5. Parse entry date: "מיידי" → `None`. DD/MM/YYYY → "YYYY-MM-DD"
6. Parse update date: DD/MM/YYYY → "YYYY-MM-DD" → `posted_date`
7. Construct `direct_url`: `https://www.homeless.co.il/rent/viewad,{ad_id}.aspx`
8. Build `location_text`: `"{street}, {city}"` (omit street if empty)
9. Build `district`: neighborhood (or city if neighborhood empty)
10. Build `title`: `"{property_type} {rooms} חדרים ב{city}"` (Hebrew)
11. Build `summary`: `"{property_type} · {rooms} חדרים · קומה {floor}"`

Return `ScrapedListing(...)` with:
- `source=self.source_id`
- `source_listing_id=ad_id`
- `source_search_url=self.search_url`
- `title=title`
- `price=price`
- `currency=self.city.currency` (→ "ILS")
- `country=self.city.country` (→ "IL")
- `available_from=available_from`
- `location_text=location_text`
- `district=district`
- `transit_match_lines=[]`
- `roommate_signal=""`
- `vegan_signal=""`
- `summary=summary`
- `direct_url=direct_url`
- `url_status="direct"`
- `recovery_query=f"site:homeless.co.il {title[:40]}"`
- `posted_date=posted_date`

**Method: `_has_next_page(soup)` → `bool`**

Check if `<div id="paging">` contains a link to the next page number.

**Method: `_parse_price(text)` → `int | None`**

Strip whitespace, remove "₪", remove commas, convert to int. Return None if empty or non-numeric.

**Method: `_parse_date(text)` → `str | None`**

If "מיידי" → return None. If DD/MM/YYYY → return "YYYY-MM-DD". Otherwise return None.

## 4. Error Handling

| Error | Expected Behavior |
|-------|------------------|
| Playwright not installed | `fetch_listings()` returns `[]` (ImportError caught) |
| Cloudflare challenge not resolved | Page content has no listing rows → returns `[]` |
| Network failure / timeout | Return listings collected so far (partial results OK) |
| Unparseable row (missing cells) | Skip row, continue to next |
| Price not parseable | `price=None` |
| Date not parseable | `available_from=None` or `posted_date=None` |
| Empty listing table (no rows) | Return `[]` |
| Browser crash | Exception caught, return `[]` |
