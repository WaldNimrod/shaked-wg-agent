## 2. Technical Specification

### 2.1 Source Registration — `data/sources.json`

**What to implement:**

Add a new source entry for Homeless.co.il to the existing `sources.json` array:

```json
{
  "source_id": "homeless",
  "label": "homeless.co.il",
  "base_url": "https://www.homeless.co.il",
  "scraper_class": "shaked_wg_agent.scrapers.homeless.HomelessScraper",
  "requires_playwright": true,
  "notes": "Israeli rental classifieds. Playwright required — Cloudflare managed challenge blocks plain HTTP. Pardes Hanna region via area ID/search params.",
  "city_params": {
    "pardes-hanna-region": {
      "search_url": "",
      "connection_method": "search_params",
      "enabled": true
    }
  }
}
```

**Note:** The exact `search_url` will be determined by the scraper implementation (WP002). The builder should set it based on how the Homeless search works (URL parameters for area/region). If the scraper constructs URLs programmatically from the bounding box, `search_url` can be the base search page URL.

**Acceptance criteria:**

- AC-01: `sources.json` contains an entry with `source_id: "homeless"`.
- AC-02: `scraper_class` is `"shaked_wg_agent.scrapers.homeless.HomelessScraper"` (FQN).
- AC-03: `city_params` has a `"pardes-hanna-region"` key with `enabled: true`.
- AC-04: Existing 3 Swiss sources remain unchanged.

### 2.2 Israeli Search Profile — `data/profiles/pardes-hanna.json`

**What to implement:**

Create a new profile file:

```json
{
  "profile_id": "pardes-hanna",
  "profile_name": "Pardes Hanna Region Search",
  "city_id": "pardes-hanna-region",
  "move_in_from": "2026-09-01",
  "budget_min": 1500,
  "budget_max": 5000,
  "preferred_roommate_age": "any",
  "rental_duration": "permanent",
  "diet": "",
  "smoking_policy": "",
  "transit_lines": [],
  "custom_tags": [],
  "language_policy": {
    "primary_listing_language": "he",
    "translation_required": false,
    "preserve_source_text": true
  },
  "retention_days": 30,
  "enabled_sources": ["homeless"],
  "notifications": {
    "digest_max_listings": 10,
    "min_score_threshold": 0,
    "channels": []
  }
}
```

**Design decisions:**

- `budget_min: 1500` / `budget_max: 5000` — ILS range for Pardes Hanna region rentals (1-3 bedroom apartments). Reasonable for the area.
- `preferred_roommate_age: "any"` — Decision D2 expands beyond WG to all rentals.
- `diet: ""` / `smoking_policy: ""` — Not filtering on these initially.
- `transit_lines: []` — Israeli transit data not yet mapped to listing format.
- `min_score_threshold: 0` — Show all results initially (no scoring filter).
- `primary_listing_language: "he"` — Hebrew listings.

**Acceptance criteria:**

- AC-05: File `data/profiles/pardes-hanna.json` exists and is valid JSON.
- AC-06: `profile_id` is `"pardes-hanna"`.
- AC-07: `city_id` is `"pardes-hanna-region"` (matches WP001 city definition).
- AC-08: `budget_min` and `budget_max` are integers in ILS (not CHF).
- AC-09: `enabled_sources` is `["homeless"]`.
- AC-10: `language_policy.primary_listing_language` is `"he"`.

### 2.3 End-to-End Verification

**What to verify:**

After WP001 (city), WP002 (scraper), and this WP are all complete, the full pipeline should work:

1. `load_config("pardes-hanna")` returns a valid `ProjectConfig` with city `country="IL"`, `currency="ILS"`.
2. The runner resolves `HomelessScraper` via FQN from `sources.json`.
3. The scraper fetches listings and returns `list[ScrapedListing]` with `currency="ILS"`, `country="IL"`.
4. The scorer scores listings using `LOCALE_IL` vegan keywords (Hebrew).
5. Results are saved to `data/listings.json`.

**Acceptance criteria:**

- AC-11: `load_config("pardes-hanna")` succeeds without error.
- AC-12: `cfg.city.country == "IL"` and `cfg.city.currency == "ILS"`.
- AC-13: `cfg.active_sources` includes the homeless source with FQN class path.
- AC-14: Running the search pipeline produces at least 1 listing (assuming Homeless has Pardes Hanna area listings).
- AC-15: Each listing dict has `country: "IL"` and `currency: "ILS"` keys.

## 4. Error Handling

| Error | Expected Behavior |
|-------|------------------|
| Homeless source not in `sources.json` | Runner skips it silently (source not found for city) |

| No listings found for Pardes Hanna | Scraper returns `[]`, run record shows 0 new listings — not an error |
